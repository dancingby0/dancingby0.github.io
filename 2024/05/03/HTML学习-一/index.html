HTML学习(一)

2024/5/3 前端 HTML

什么是HTML?

HTML(Hyper Text Markup Language) 超文本标记语言
非编程语言,而是标记语言

基本格式

<!DOCTYPE html>
<!-- 声明为HTML5文档 -->
<html>
<head>
<title>这里是标题</title>
</head>
<body>
    <!-- 这是注释 -->
    <!-- body标签表示页面的主要内容-->
    <p>这里是段落</p>

</body>
<!-- 注意,所有标签尽可能用/关闭-->
</html>

一些标签
<p>段落标签,表示一段
<h1>-<h6>表示标题
<a href=”链接”>链接文本</a>
<b>表示加粗
<s>表示删除线
<u>表示下划线
<i>表示斜体
<sup>表示上标
<sub>表示下标
<pre>表示以源格式输出
<em>表示强调
<strong>表示加重语气
<q>表示内联引用
<blockquote>表示块引用
<hr>表示分割线
<br>表示换行
<span>表示范围内生效(用css)
<ul>表示无序列表
<ol>表示有序列表
<li>表示列表中的元素
列表可以嵌套
<div>表示分割,并不会表示任何内容,更用于组织内容,如使用css或id等.
<section>表示部分
应当注意的是,一篇网页应当只有一个main,不同章节用section分割,更小的部分用div.

<html>头内可以有lang属性,标识网页的语言.
